Giblogをはじめよう

Giblogは、gitで管理できるWebサイト・ブログを作成するツールです。日本では、CMSとも呼ばれていますね。Giblogで作ったWebサイトをGitで管理すると、複数のWebサイト運営が楽になります。

GiblogをWebサイト制作に採用する12の理由

  • すべてのページをgitで管理
  • 高度なSEO対策
  • レスポンシブデザインでスマートフォン対応
  • お問い合わせフォームのサポート
  • 高速な再構築と再構築の自動化
  • 強固なセキュリティ
  • 記事の検索・置換をLinuxコマンドで可能
  • Windows、Mac OS、LinuxなどマルチOS対応
  • サイトをPerlで自由にカスタマイズできる
  • ハイパフォーマンスなWebページ
  • Github Pagesを使って完全無料でサイト公開
  • フォーラムによるユーザーサポート

環境構築

Perlをまったく知らなくても、手順を実行するだけで、Webサイト・ブログの作成が可能です。サイトの機能を変更・追加したい場合に、プログラミング言語であるPerlを使って自由にカスタマイズできます。「プログラムはわからないや」という方のために、Perlのサンプルは、できるかぎりコピー＆ペーストで動くようにしています。30分程度の短い手順です。
